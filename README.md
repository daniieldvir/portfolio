# Personal Portfolio

Welcome to my personal portfolio project! This application is built with **Angular** to showcase my projects, skills, and experience as a software developer.

## Built With

*   [Angular](https://angular.dev/) (v21) - The web framework used
*   [NGXS](https://www.ngxs.io/) - State management pattern + library for Angular
*   [NgIcons](https://ng-icons.github.io/ng-icons/) - SVG icon library for Angular
*   [Vitest](https://vitest.dev/) - Blazing fast unit test framework
*   [SCSS](https://sass-lang.com/) - Styling

## Prerequisites

Before you begin, ensure you have the following installed on your local machine:

*   [Node.js](https://nodejs.org/) (version 18.x or higher is recommended)
*   [npm](https://www.npmjs.com/) (Node Package Manager)

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/daniieldvir/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **To start the development server:**
    ```bash
    npm start
    ```
    Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Running Unit Tests

To execute unit tests using Vitest, run:
```bash
npm test
```
